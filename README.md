# Plataforma Digital para ONGs - Entrega II: Estilização e Leiautes

## Descrição do Projeto
Este repositório contém a segunda entrega do projeto de desenvolvimento web, focada na estilização e aplicação de leiautes responsivos à estrutura HTML da Entrega I. O objetivo é transformar a estrutura básica em uma interface visualmente agradável, profissional e acessível, utilizando CSS3 puro.

## Tecnologias Utilizadas
- HTML5 (Estrutura da Entrega I)
- CSS3 (Estilização, Flexbox, Grid, Variáveis CSS)
- JavaScript (Apenas para a funcionalidade básica do menu mobile)

## Estrutura de Arquivos
- `index.html`: Página principal com a estrutura HTML.
- `css/`: Pasta com todos os arquivos de estilo.
  - `styles.css`: Arquivo principal que importa os módulos.
  - `css/modules/`: Módulos de Design System, Layout, Componentes e Responsividade.
- `js/main.js`: Script simples para o menu hambúrguer.

## Requisitos
Para visualizar o projeto, basta abrir o arquivo `index.html` em qualquer navegador moderno.
